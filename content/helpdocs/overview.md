# 🏠 Overview

Welcome to **{{brand.name}}** — your autonomous AI-powered factory that orchestrates AI agents through structured pipelines to produce real deliverables.

---

## 💡 What is {{brand.name}}?

{{brand.name}} lets you build a **digital crew** of AI agents, organize them into **pipelines**, and run **sprints** that produce real output — code, documents, research, analysis, or anything your agents are designed for.

> **You bring the vision. Your crew brings the execution.**

---

## 🧩 Core Concepts

| Concept | What it is |
|---------|-----------|
| 🏭 **Factory** | Your workspace — groups agents, pipelines, and projects |
| 🤖 **Agent** | An AI persona with a defined role, capabilities, and tools |
| 🔄 **Pipeline** | A multi-phase workflow that orchestrates agents in sequence |
| 📁 **Project** | A brief + pipeline + configuration = ready to run |
| 🚀 **Sprint** | One execution of a pipeline for a project |
| 🛒 **Marketplace** | Community store for agents and factories |

---

## 🗺️ How It Works

```
1. 🏭  Create a Factory (your workspace)
2. 🎨  In the Studio:
       → Create Agents (your AI crew)
       → Build Pipelines (the workflow)
       → Set up Projects (brief + pipeline + config)
3. 🚀  In the Office:
       → Start a Sprint
       → Agents execute sequentially
       → Approve gates, review output
       → Ship the result
```
