# ✅ Prerequisites

Before using {{brand.name}}, you need a few accounts and tools set up. What you need depends on whether you run **locally** or in the **cloud**.

---

## 📋 Quick Checklist

| Requirement | Local Mode | Cloud Mode |
|-------------|:----------:|:----------:|
| {{brand.name}} Account | ✅ Required | ✅ Required |
| Trigger.dev Account | ✅ Required | ✅ Required |
| LLM API Keys | ⚡ Optional* | ✅ Required |
| Repository (GitHub/GitLab) | 📎 Optional | 📎 Optional |
| Supabase (cloud storage) | 📎 Optional | ✅ Required |
| Node.js 20+ on your machine | ✅ Required | ❌ Not needed |
| CLI tools installed | ⚡ Optional | ❌ Not needed |

> \* In local mode, you can use **CLI subscriptions** (e.g., Claude Pro) instead of API keys.

---

## 🔗 Detailed Setup

Each prerequisite has its own guide — expand the subtopics below for step-by-step instructions:

- **{{brand.name}} Account** — Sign up and create your organization
- **Trigger.dev** — Set up orchestration for pipelines
- **Repositories** — Connect GitHub or GitLab for sprint output
- **Cloud Orchestration** — Supabase for storage and database
- **Local Orchestration** — Machine requirements for local mode
