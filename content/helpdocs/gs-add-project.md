# ➕ Add a Project to the Office

Projects appear in the Office queue once they have a pipeline assigned and a briefing written.

---

## 📌 From Studio

1. Go to **Studio → Projects**
2. Create a project with name + brief
3. Assign a pipeline in **Project Settings**
4. The project automatically appears in the Office queue

## 📌 From Office

1. Click **+ New project** in the Office
2. Fill in name and brief
3. The project is created and ready to run

---

## 📋 Project Cards

Each project card in the Office shows:

| Element | Description |
|---------|-------------|
| 🎯 **Status badge** | Current state (Ready, Running, Waiting, etc.) |
| 🤖 **Active agent** | Which agent is currently executing (if running) |
| 🔢 **Sprint number** | Current sprint iteration |
| ⚠️ **Last error** | Error message if the last sprint failed |
| 🔒 **Lock icon** | Prevent accidental actions on the project |
| ⚙️ **Settings** | Open project settings |
| ▶️ **Start Sprint** | Launch the sprint modal |
