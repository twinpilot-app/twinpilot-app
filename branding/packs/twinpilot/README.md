# Twin Pilot — Branding Kit

Visual identity system for **Twin Pilot** (twinpilot.app) — the AI-powered software factory.

## Brand Essence

Twin Pilot is about **duality**: human + AI working in parallel, like a co-pilot. The visual identity reflects this through mirrored geometry, dual-tone gradients, and a dark-first aesthetic inspired by developer tools (Linear, Vercel, Supabase).

## Design Principles

1. **Dark-first** — all primary interfaces use dark theme
2. **Dual gradients** — brand gradient flows from electric violet to cyan
3. **Minimal chrome** — content breathes, sparse borders, subtle surfaces
4. **Monospace accents** — code-like elements for technical credibility
5. **Glow** — subtle glow effects on interactive elements

## Typography

- **Headings/Wordmark**: Inter (700, 800)
- **Body**: Inter (400, 500, 600)
- **Code/Mono**: JetBrains Mono

## Logo Usage

- Minimum clear space: half the mark width on all sides
- Never recolor outside the approved palette
- Never rotate, stretch, or add effects
- Use white variant on dark, black variant on light
- The mark works standalone at 24px+ sizes
